# Tiny Notes Demo

This folder is a simple target for testing WebChat prompts.

Open `notes.js` in Cursor, run `WebChat: Send Prompt to Browser`, choose a provider, and the browser extension should place a prompt into the active web chat input.

Suggested user request to paste at the top of the file or include in the prompt:

```text
Refactor this tiny notes module to support tags, case-insensitive search, and a simple export function. Keep it dependency-free.
```
